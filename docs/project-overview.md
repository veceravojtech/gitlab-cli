# Project Overview - gitlab-cli

## Summary

**gitlab-cli** is a command-line tool for managing GitLab merge requests with automated rebase and merge workflows. It streamlines common MR operations with CI-aware merging, animated progress feedback, and flexible output formats.

## Key Features

| Feature | Description |
|---------|-------------|
| **MR Management** | List, show, create, rebase, merge merge requests |
| **Auto-Rebase** | Automatically rebase and retry when merging |
| **CI-Aware Merging** | Waits for pipelines before merging |
| **Activity Tracking** | Log user activities with task extraction |
| **Multiple Outputs** | Table, JSON, CSV output formats |
| **Progress Feedback** | Animated status during long operations |

## Installation

### From Source
```bash
git clone https://github.com/veceravojtech/gitlab-cli.git
cd gitlab-cli
./install.sh
```

### Configuration
```bash
# Environment variables (recommended)
export GITLAB_URL="https://gitlab.example.com"
export GITLAB_TOKEN="glpat-xxxxxxxxxxxx"

# Or config file
gitlab-cli config init
```

## Usage Examples

```bash
# List your open MRs
gitlab-cli mr list --mine

# Merge with auto-rebase
gitlab-cli mr merge 456 --auto-rebase --max-retries 5

# View activity for current month
gitlab-cli activity list --group-by-task

# Export activity as CSV
gitlab-cli activity list --format csv > activity.csv
```
